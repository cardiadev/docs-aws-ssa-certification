# Course Overview

Welcome to the **AWS Solutions Architect Associate** certification course at KodeKloud! This course is split into two major sections:

- **Services Section:** Led by **Sanjeev Thiyagarajan**, covering AWS service categories at a technical level.
- **Design Section:** Led by **Michael Forrester**, covering how to architect in the cloud using the pillars of the Well-Architected Framework.

## Unique Features of This Course

| Feature | Description |
|---|---|
| **Browser-based Labs** | No need for your own AWS account. Access real AWS environments interactively in your browser. |
| **Engaging Lectures** | Clear, captivating lessons covering exactly what you need to master the certification. |
| **Hands-on Demos** | Real-world examples of AWS services being implemented and running in action. |
| **Reinforcement Quizzes** | Strategic checkpoints scattered throughout to help you track your progress. |
| **Gamification** | Drag and drop services into architectural setups to learn in a playful, intuitive way. |
| **Mock Exams** | One at the beginning (to verify your baseline) and one at the end (to measure growth). |

## Course Structure

### Mega-Section 1: Meet Your Services
Categories covered:
- Networking
- Storage
- Compute
- Database
- Application Integration
- Data and Machine Learning
- Migration and Transfer
- Management and Governance
- Security

### Mega-Section 2: Design for X
The four pillars of the Well-Architected Framework:
- **Security** — Encryption, access control, and best practices.
- **Reliability** — Fault tolerance and high availability.
- **Performance** — Scalability and efficiency.
- **Cost Optimization** — Strategies to reduce spend without sacrificing performance.
