# Prerequisites: Are You Ready?

## Why This Matters

The **Solutions Architect Associate** course assumes you have knowledge equivalent to the **AWS Cloud Practitioner** certification. Without that foundation, you will struggle significantly.

## The Optional Mock Exam

- A **link to a Cloud Practitioner mock exam** is provided in the course.
- It is **optional**, but highly recommended if you don't hold that certification.
- It helps you determine if you're ready to begin this course.

## Decision Criteria

| Score | Recommendation |
|---|---|
| **≥ 70** | ✅ You have sufficient foundation — continue with this course |
| **< 70** | ⚠️ It is highly recommended to take the Cloud Practitioner course first |

> It is **not mandatory** to hold the certification, but without the baseline knowledge this course will be very difficult.

## Next Steps

1. Take the Cloud Practitioner mock exam.
2. If you score 70+: keep going!
3. If you don't pass: go back and take the [KodeKloud Cloud Practitioner course](https://kodekloud.com) then return here.
