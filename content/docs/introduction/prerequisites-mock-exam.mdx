## ¿Por Qué Importa Esto?

Este curso de **Solutions Architect Associate (SAA)** asume que ya tienes un nivel equivalente al de la certificación **AWS Cloud Practitioner (CCP)**. Esto significa que debes entender conceptos básicos como:

| Concepto | Ejemplo |
|---|---|
| **Regiones y AZs** | Saber qué es us-east-1, qué es una Availability Zone |
| **EC2 básico** | Qué es una instancia, tipos de instancia, AMIs |
| **S3 básico** | Qué es un bucket, objetos, permisos básicos |
| **IAM básico** | Usuarios, grupos, políticas, roles |
| **Modelo de precios** | Free Tier, On-Demand, Reserved |

<Callout type="warn" title="Sin esta base, el curso será muy difícil">
No es **obligatorio** tener la certificación CCP, pero si no entiendes estos conceptos básicos, este curso avanzará demasiado rápido y te sentirás perdido.
</Callout>

## El Mock Exam de Cloud Practitioner

El curso incluye un **mock exam de Cloud Practitioner** como punto de control. Es **opcional pero muy recomendado** si no tienes la certificación.

| Aspecto | Detalle |
|---|---|
| **Formato** | Preguntas de opción múltiple tipo CCP |
| **Propósito** | Verificar que tienes la base necesaria |
| **Obligatorio** | No, pero altamente recomendado |
| **Criterio de aprobación** | **≥70%** |

## Interpretación de tu Puntuación

| Puntuación | Resultado | Acción Recomendada |
|---|---|---|
| **≥70%** | **Aprobado** — tienes la base suficiente | Continúa con el curso SAA sin problemas |
| **50–69%** | **Límite** — conocimiento parcial | Puedes continuar, pero considera repasar los conceptos donde fallaste |
| **&lt;50%** | **Insuficiente** — necesitas refuerzo | Toma el curso de Cloud Practitioner primero |

<Callout type="info" title="El ∐70% es tu línea de decisión">
Si obtienes **70% o más**, tienes suficiente base para continuar. Si estás por debajo, no es una derrota — simplemente significa que necesitas construir la base primero con el curso de Cloud Practitioner.
</Callout>

## Flujo de Decisión

| ¿Tienes CCP? | ¿Tomaste el mock exam? | Puntuación | → Acción |
|---|---|---|---|
| **Sí** (certificación aprobada) | No necesario | — | Continúa con SAA |
| **No**, pero tengo experiencia AWS | **Sí** | ≥70% | Continúa con SAA |
| **No**, pero tengo experiencia AWS | **Sí** | &lt;70% | Toma el curso CCP primero |
| **No**, sin experiencia | **Sí** | Cualquiera | Toma el curso CCP primero |
| **No**, sin experiencia | **No** | — | Toma el curso CCP primero |
