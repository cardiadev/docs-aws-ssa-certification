# Prerrequisitos: ¿Estás Listo?

## ¿Por Qué Importa Esto?

Este curso de **Solutions Architect Associate** asume que ya tienes un nivel equivalente al de la certificación **AWS Cloud Practitioner**. Si no tienes esa base, te vas a encontrar con muchas dificultades.

## El Mock Exam Opcíonal

- Se te proporciona un **link a un mock exam** de Cloud Practitioner.
- Es **opcional**, pero altamente recomendado si no tienes la certificación.
- Sirve para descubrir si estás listo para empezar este curso.

## Criterio de Decisión

| Puntuación | Recomendación |
|---|---|
| **≥ 70** | ✅ Tienes la base suficiente — continúa con este curso |
| **< 70** | ⚠️ Se recomienda tomar el curso de Cloud Practitioner primero |

> No es **obligatorio** tener la certificación, pero si no tienes el conocimiento base, este curso será muy difícil.

## Próximos Pasos

1. Toma el mock exam de Cloud Practitioner.
2. Si pasas con 70+: ¡sigue adelante!
3. Si no pasas: regresa a tomar el [curso de Cloud Practitioner de KodeKloud](https://kodekloud.com) y luego vuelve aquí.
