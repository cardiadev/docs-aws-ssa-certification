# The SAA-C03 Exam Guide

## Purpose of the Exam

The **SAA-C03** exam validates your ability to design and review AWS solutions and architectures, optimizing across four dimensions: security, resiliency, performance, and cost.

## Target Candidate

Ideally, you have **at least one year of hands-on experience with AWS**. That said, building a project portfolio can substitute for formal work experience.

## Exam Format

| Aspect | Detail |
|---|---|
| Total questions | 65 (50 scored + 15 unscored pilot questions) |
| Time limit | 130 minutes |
| Question types | Multiple choice / Multiple response |
| Passing score | 720 / 1000 |
| Practice target | Aim for 90%+ on mock exams |

## Exam Domains

| Domain | Content | Weight |
|---|---|---|
| 1 | Designing Secure Architectures | 30% |
| 2 | Designing Resilient Architectures | 26% |
| 3 | Designing High-Performing Architectures | 24% |
| 4 | Designing Cost-Optimized Architectures | 20% |

## Data Pipelines Note

Despite being an architecture exam, it includes questions on **designing data ingestion and transformation solutions**, so ML and analytics services (Glue, Kinesis, Athena, etc.) are part of the scope.
