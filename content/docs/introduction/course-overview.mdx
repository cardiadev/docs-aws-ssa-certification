# Visión General del Curso

Bienvenidos al curso de certificación **AWS Solutions Architect Associate** en KodeKloud. Este curso está dividido en dos grandes secciones:

- **Sección de Servicios:** Guiada por **Sanjeev Thiyagarajan**, cubre los servicios de AWS a nivel técnico.
- **Sección de Diseño:** Guiada por **Michael Forrester**, cubre cómo diseñar arquitecturas usando los pilares del Well-Architected Framework.

## Características Únicas del Curso

| Característica | Descripción |
|---|---|
| **Labs en el Navegador** | Sin necesidad de cuenta propia. Accede a entornos AWS reales directamente desde el navegador. |
| **Clases Claras** | Lecciones diseñadas para ser informativas y captivantes, cubriendo exactamente lo que necesitas para la certificación. |
| **Demos Prácticas** | Ejemplos del mundo real de servicios AWS funcionando en acción. |
| **Quizzes de Refuerzo** | Puntos de control estratégicos para medir tu comprensión a lo largo del curso. |
| **Gamificación** | Arrastra y suelta servicios en configuraciones arquitectónicas para aprender de forma interactiva. |
| **Mock Exams** | Un examen al inicio (para verificar tu base) y uno al final (para medir tu progreso). |

## Estructura del Curso

### Mega-Sección 1: Conoce tus Servicios
Categorías cubiertas:
- Redes (Networking)
- Almacenamiento (Storage)
- Computación (Compute)
- Bases de Datos (Database)
- Integración de Aplicaciones
- Datos y Machine Learning
- Migración y Transferencia
- Gestión y Gobernanza
- Seguridad

### Mega-Sección 2: Diseño para X
Los cuatro pilares del Well-Architected Framework:
- **Seguridad** — Cifrado, control de accesos, buenas prácticas.
- **Fiabilidad** — Tolerancia a fallos, alta disponibilidad.
- **Rendimiento** — Escalabilidad, eficiencia.
- **Optimización de Costos** — Estrategias para reducir gastos sin perder performance.
