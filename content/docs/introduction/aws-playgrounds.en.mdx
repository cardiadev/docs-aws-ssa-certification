# The KodeKloud AWS Playgrounds

## What Are Playgrounds?

**KodeKloud Playgrounds** (available at KodeKloud.com/Playgrounds) are sandbox environments where you can interact with AWS services hands-on **without personal billing charges**.

## Key Advantages

- **No personal credit card**: You won't be billed for anything.
- **Time-limited and safe**: Sessions typically last up to 3 hours.
- **Broad service coverage**: Supports most SAA exam services: EC2, RDS, SNS, KMS, VPCs, security groups, key pairs, etc.

### Services Not Available

Some DevOps services are not supported in Playgrounds, such as **CodePipeline** and **CodeDeploy**.

## Workflow

1. Start a lab on KodeKloud.
2. Receive a temporary **Account ID**, IAM Username, and Password.
3. Log into the AWS console with those credentials.
4. Explore, launch resources, and practice freely in the sandboxed environment.

## When to Use Them

Open a Playground whenever you encounter a syllabus service you don't fully understand. Hands-on practice consolidates learning far faster than reading alone.
